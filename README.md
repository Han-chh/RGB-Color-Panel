# RGB Color Panel

## Description

An RGB color palette application built in Java using Swing. Allows users to input RGB values and visualize the resulting color.

## Features

- RGB value input fields (0-255)
- Real-time color visualization
- Color display panel
- Reset functionality
- Input validation
- Keyboard navigation (Enter to move between fields)

## Requirements

- Java Development Kit (JDK) 8 or higher

## Run

```bash
java -jar RGB-Color-Panel.jar
```

## Usage

1. Launch the application
2. Enter RGB values in the respective fields (0-255)
3. Click "generate" or press Enter in the Blue field to display the color
4. Use "Reset" to clear all fields and reset the display

## Project Structure

- `src/com/rgbcolorpanel/Main.java`: Main application window
- `src/com/rgbcolorpanel/RGBColorPanel.java`: RGB color selection panel
- `bin/`: Compiled class files
